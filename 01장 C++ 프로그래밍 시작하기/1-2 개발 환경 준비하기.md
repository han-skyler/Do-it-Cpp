# 비주얼 스튜디오 설치하기

</aside>

비주얼 스튜디오(Visual Studio)는 전 세계에서 많은 개발자가 사용하는 통합 개발 환경이다. 비주얼 스튜디오를 시용하면 C++ 언어로 작성된 코드를 컴파일하고 디버깅할 수 있으며 개발을 돕는 여러 가지 편리한 기능도 이용할 수 있다.

워크로드를 선택하는 화면이 나오면 [C++를 사용한 데스크탑 개발]을 선택하고, 오른쪽의 설치 세부 정보에서 [v143 빌드 도구용 C++모듈(x64/86 - 실험적)]에 체크 표시한다. 이는 C++20에서 추가된 모듈을 사용하기 위한 설정이다

</aside>

# 실습 파일 내려 받기

</aside>

프로그래밍 언어를 공부할 때는 코드를 키보드로 직접 작성해 보아야 빨리 익숙해진다. 그러나 코드가 너무 길거나 본인이 작성한 코드를 검증된 코드와 비교해보고 싶다면 저자가 제공하는 소스 파일을 참고할 수 있다.

[https://github.com/mystous/DoItCPP](https://github.com/mystous/DoItCPP)

이 책의 실습용 소스 파일은 모두 저자의 깃허브 저장소로부터 내려받아 실행할 수 있다.

<aside>

# 컴파일 옵션 선택하기

</aside>

이 책에서 모든 코드는 윈도우 10에서 비주얼 스튜디오 2022 커뮤니티 버전으로 테스트했다. 그리고 C++20 표준을 사용해 컴파일 했다. 비주얼 스튜디오에서 언어 표준을 확인할 때는 메뉴에서 [프로젝트 → 프로젝트명 속성]을 선택한다.

프로젝트 속성 창이 열리면 왼쪽에서 [구성 속성 → 일반]을 클릭한다. 여기서 [C++ 언어 표준]의 속성값을 [ISO C++ 표준(/std::c++20)]으로 설정한다. 만약 실습에 따라 언어 표준을 바꿔야 한다면 여기서 변경하면 된다.

이 책의 후반부에서는 모듈을 사용하는 실습도 진행하므로 모듈을 사용하는 설정도 해줘야 한다. [구성 속성 → C/C++ → 언어]에서 [실험용 C++ 표준 라이브러리 모듈 사용] 속성값을 [예(/experimental:module)]로 설정한다.

<aside>

# macOS에서 C++ 개발 환경

</aside>

macOS에서 C++ 개발 환경을 갖출 수도 있다. 그러나 macOS용 비주얼 스튜디오는 C++ 컴파일을 지원하지 않고 C# 컴파일만 지원한다. 그러므로 macOS에서는 다른 도구를 사용해야 한다.

- 엑스코드(Xcode)
- 비주얼 스튜디오 코드

단, 비주얼 스튜디오 코드에서 C++을 사용하려면 별도의 확장 패키지를 설치해야 한다. 자세한 사항은 공식 문서에서 확인할 수 있다.

[https://code.visualstudio.com/docs/languages/cpp](https://code.visualstudio.com/docs/languages/cpp)
