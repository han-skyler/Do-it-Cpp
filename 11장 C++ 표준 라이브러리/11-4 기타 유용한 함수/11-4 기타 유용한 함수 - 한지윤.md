# 난수 생성

### 난수(Random Number)의 개념

- **난수**는 정의된 범위에서 무작위로 추출된 임의의 수를 의미합니다.
- 다음에 나올 값을 누구도 확신할 수 없어야 합니다.
- 컴퓨터 프로그래밍에서 난수는 주로 **시드(seed)**라는 시작 숫자를 이용하며, 현재 시각을 시드값으로 많이 사용합니다.

### C/C++에서의 난수 생성 (과거)

- `rand()`: 난수 생성 패턴을 한 개로 설정합니다.
- `srand()`: 난수 생성 패턴을 여러 개로 설정합니다.
- 이 함수들은 `<cstdlib>` 헤더 파일에 정의되어 있습니다. (강의 자료에는 `<random>`으로 기재되어 있으나, `rand`와 `srand`는 전통적으로 `<cstdlib>`에 포함됨)
- 매크로 상수 `RAND_MAX`는 `0x7fff`로 정의되며, 이는 난수의 범위가 **0부터 32,767**까지임을 의미합니다.
- 이 범위는 난수가 균등하게 분포되지 않을 수 있다는 단점이 있습니다.

### C++11 이후의 고품질 난수 생성

- C++11부터는 `<random>` 헤더 파일을 통해 **고품질 난수 생성기**와 **분포 클래스**를 제공합니다.
- 이를 통해 난수의 형식, 범위, 분포, 형태 등을 세밀하게 조절할 수 있습니다.
- **`std::mt19937`**: 32비트 메르센 트위스터(Mersenne Twister) 난수 생성 엔진.
- **`std::mt19937_64`**: 64비트 메르센 트위스터 난수 생성 엔진.

### 난수 생성 예시 (`std::mt19937_64` 사용)

```cpp
#include <iostream>
#include <random>

using namespace std;

int main() {
    mt19937_64 int_rand; // 64비트 난수 생성기 객체 생성
    for (int i = 0; i < 10; i++) {
        cout << int_rand() << endl; // 난수 10개 생성 및 출력
    }
    return 0;
}
```

- *주의: 시드값을 사용하지 않으면 실행할 때마다 같은 값이 나올 수 있습니다.*

### 시드값으로 난수 생성 (`<chrono>` 활용)

- `std::chrono::system_clock::now()`를 사용하여 현재 시각을 얻고, 이를 밀리초 단위로 변환하여 시드값으로 사용합니다.

```cpp
#include <iostream>
#include <random>
#include <chrono> // 시간 관련 기능을 위한 헤더

using namespace std;

int main() {
    // 시드값 사용
    auto curTime = chrono::system_clock::now();
    auto duration = curTime.time_since_epoch();
    auto millis = chrono::duration_cast<chrono::milliseconds>(duration).count();

    mt19937_64 nit_rand(millis); // 현재 시각을 시드값으로 사용하여 난수 생성기 초기화
    for (int i = 0; i < 10; i++) {
        cout << nit_rand() << endl;
    }
    return 0;
}
```

- *이처럼 시드값을 이용하면 실행할 때마다 다른 값을 만들 수 있습니다.*

### 하드웨어 엔트로피를 이용한 난수 생성 (`std::random_device`)

- **하드웨어 엔트로피**: 시스템에서 발생하는 무작위성의 정도를 나타냅니다 (마우스 움직임, 키보드 입력, 디스크 I/O 등).
- `std::random_device` 클래스는 이러한 하드웨어 엔트로피를 활용하여 난수를 생성합니다.

```cpp
#include <iostream>
#include <random>

using namespace std;

int main() {
    random_device rng; // 하드웨어 엔트로피 기반 난수 생성기 객체 생성
    for (int i = 0; i < 10; i++) {
        auto result = rng();
        cout << result << endl;
    }
    return 0;
}
```

- *단점: `random_device`는 `mt19937` 엔진보다 느리므로, 난수 생성이 많을 때는 `mt19937`을 사용하고 `random_device`는 엔진의 시드값 생성에만 사용하는 것이 좋습니다.*

### `<random>` 헤더에서 제공하는 다른 난수 생성 엔진

- `linear_congruential_engine`: 선형 합동 난수 엔진
- `mersenne_twister_engine`: 메르센 트위스터 난수 엔진
- `subtract_with_carry_engine`: 감산 캐리 난수 엔진

### 완벽한 난수와 의사 난수

- **완벽한 난수**: 예측 불가능하며 어떠한 패턴도 갖지 않아야 합니다.
- **의사 난수(pseudo-random number)**: 컴퓨터에서 사용되는 대부분의 난수.
    - 특정한 알고리즘을 기초로 생성됩니다.
    - 초깃값(시드값)이 같으면 같은 순서로 생성되므로 예측 가능합니다.
    - 오랫동안 사용하면 주기적으로 반복되는 패턴이 나타날 수 있습니다.
- **보완 방법**: 더 복잡하고 예측 어려운 알고리즘 사용, 물리적 현상을 활용하는 하드웨어 난수 발생기(양자역학 기반) 사용.

## 수학 함수

- C++ 표준 라이브러리는 다양한 수학 함수를 제공하며, `<cmath>` 헤더 파일을 포함하여 사용할 수 있습니다.
- **C++20부터 `numbers::pi` 상수를 사용할 수 있습니다.**

### 자주 사용하는 수학 함수

| 함수 | 설명 |
| --- | --- |
| `abs` | 절댓값 반환 |
| `sqrt` | 제곱근 반환 |
| `pow` | 거듭제곱 계산 |
| `exp` | 지수 함수 ($e^x$) 계산 |
| `log` | 자연 로그 계산 |
| `sin`, `cos`, `tan` | 삼각 함수 계산 |
| `asin`, `acos`, `atan` | 역삼각 함수 계산 |
| `ceil` | 올림 계산 |
| `floor` | 내림 계산 |
| `round` | 반올림 계산 |

### 수학 함수 활용 예시

```cpp
#include <iostream>
#include <cmath>
#include <numbers> // C++20부터 pi 상수 제공

using namespace std;

int main() {
    double x = 2;
    double y = 3;
    cout << "pow(x, y) = " << pow(x, y) << endl; // x의 y승: 2^3 = 8
    cout << "sqrt(x) = " << sqrt(x) << endl;     // x의 제곱근: sqrt(2) = 1.41421
    cout << "log(x) = " << log(x) << endl;       // x의 자연로그: ln(2) = 0.693147

    x = 2.847;
    y = -3.234;
    cout << "ceil(x) = " << ceil(x) << ", ceil(y) = " << ceil(y) << endl; // 올림: 3, -3
    cout << "floor(x) = " << floor(x) << ", floor(y) = " << floor(y) << endl; // 내림: 2, -4
    cout << "round(x) = " << round(x) << ", round(y) = " << round(y) << endl; // 반올림: 3, -3
    cout << "abs(x) = " << abs(x) << ", abs(y) = " << abs(y) << endl;     // 절댓값: 2.847, 3.234

    cout << "PI = " << numbers::pi << endl; // 원주율
    cout << "sin(PI/3) = " << sin(numbers::pi / 3) << endl; // sin(60도) = 0.866025
    cout << "cos(PI/3) = " << cos(numbers::pi / 3) << endl; // cos(60도) = 0.5
    cout << "tan(PI/3) = " << tan(numbers::pi / 3) << endl; // tan(60도) = 1.73205
    return 0;
}
```

## 복사 함수

### 얕은 복사(Shallow Copy)와 깊은 복사(Deep Copy)

- **얕은 복사**:
    - 주솟값을 복사합니다.
    - 원본과 복사본이 동일한 메모리 공간을 참조하므로, 원본 값이 바뀌면 복사본도 바뀐 값을 참조합니다.
- **깊은 복사**:
    - 실제 값을 새로운 독립적인 메모리 공간에 복사합니다.
    - 원본 값이 바뀌어도 복사한 값에는 영향이 없습니다.

### `std::copy` 함수를 이용한 깊은 복사

- 반복문으로 직접 깊은 복사를 구현하면 코드가 길어지고 성능 문제가 발생할 수 있습니다.
- 표준 라이브러리는 깊은 복사를 위한 `std::copy` 함수를 제공하여 이를 보완하고 성능을 확보합니다.
- `std::copy`는 `<algorithm>` 헤더 파일에 정의되어 있습니다.

### `copy` 함수 원형

```cpp
template <class _InIt, class _OutIt>
_CONSTEXPR20 _OutIt copy(_InIt _First, _InIt _Last, _OutIt _Dest)
```

- `_First`부터 `_Last` 전까지의 모든 원소를 `_Dest`부터 시작하는 곳에 복사합니다.

### `copy` 함수 활용 예시 (Person 구조체 벡터 복사)

```cpp
#include <iostream>
#include <vector>
#include <algorithm> // copy 함수를 위한 헤더
#include <string>    // string 사용을 위한 헤더
#include <iterator>  // back_inserter를 위한 헤더

using namespace std;

// 사람 정보를 담는 구조체
struct Person {
    string name;   // 이름
    int age;       // 나이
    float height;  // 키
    float weight;  // 몸무게
};

// 벡터에 저장된 사람 정보 출력 함수
void print_person_all(vector<Person>& vec) {
    for (vector<Person>::iterator it = vec.begin(); it != vec.end(); it++) {
        cout << "이름: " << it->name << "\\t > "
             << "나이: " << it->age << ", "
             << "키: " << it->height << ", "
             << "몸무게: " << it->weight << endl;
    }
}

int main() {
    // Person 구조체 배열 생성
    Person p[5] = {
        {"Brain", 24, 180, 70},
        {"Jessica", 22, 165, 55},
        {"James", 30, 170, 65},
        {"Tom", 12, 155, 46},
        {"Mary", 18, 172, 62}
    };

    // from_vector에 Person 배열 순서대로 넣기
    vector<Person> from_vector;
    for (int i = 0; i < 5; ++i) {
        from_vector.push_back(p[i]);
    }

    cout << "------- from_vector----------" << endl;
    print_person_all(from_vector);
    cout << endl;

    // to_vector에 from_vector의 원소를 '깊은 복사' 수행
    vector<Person> to_vector;
    // back_inserter는 대상 컨테이너의 끝에 원소를 추가하는 이터레이터
    copy(from_vector.begin(), from_vector.end(), back_inserter(to_vector));

    // 복사 후 to_vector 출력
    cout << "------- to_vector----------" << endl;
    print_person_all(to_vector);
    cout << endl;

    // from_vector의 첫 번째 원소 수정
    from_vector[0].name = "Chris";
    from_vector[0].age = 5;
    from_vector[0].height = 110;
    from_vector[0].weight = 20;

    // 수정 후 from_vector 출력
    cout << "-------- from_vector---------" << endl;
    print_person_all(from_vector);
    cout << endl;

    // to_vector 출력 (원본 수정에 영향받지 않음을 확인)
    cout << "------- to_vector----------" << endl;
    print_person_all(to_vector);
    cout << endl;

    return 0;
}
```

- **실행 결과 확인**: `from_vector`의 첫 번째 원소를 수정한 후에도 `to_vector`의 원소는 변경되지 않고 초기 복사된 값을 그대로 유지합니다.
- 이는 `std::copy` 함수가 **깊은 복사**를 수행하여 원본과 독립적인 새로운 메모리 공간에 값을 복사했음을 보여줍니다.
- `std::copy` 함수를 활용하면 코드가 간결해지고 성능을 확보할 수 있습니다.
