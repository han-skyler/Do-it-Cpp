# 11-2 문자열 라이브러리

# 표준 문자열 라이브러리 `std::string`

C++에서 문자열을 다루기 위한 표준 라이브러리로, `<string>` 헤더 파일을 포함하여 사용합니다. `std::string` 객체는 직접 문자열을 대입하거나 생성자를 통해 초기화할 수 있으며, `char` 배열처럼 개별 문자에 접근할 수 있습니다.

<aside>

### **왜 `std::string`을 사용하나?**

C언어에서는 문자열을 다룰 때 `char` 배열(예: `char myText[] = "Hello";`)을 사용했는데, 이건 좀 불편한 점이 많았습니다. 

예를 들어, 문자열의 길이를 직접 관리해야 하거나, 문자열을 합치거나 복사할 때 메모리 관리를 신경 써야 했습니다.

`std::string`은 이런 불편함을 해소해 줍니다.

- **길이 자동 관리:** 문자열이 길어지거나 짧아져도 `string`이 알아서 길이를 조절합니다.
- **쉬운 초기화:** `string str1("Hello");`처럼 생성자로 초기화하거나, `string str1 = "Hello";`처럼 `=` 기호로 바로 대입할 수 있습니다.
</aside>

# 문자열 길이 구하기 — `length`, `size`

`length()`와 `size()` 함수는 모두 문자열의 길이를 반환합니다. `std::string`의 경우 두 함수의 반환값이 항상 동일합니다.

- `length()`: 문자열의 **길이**를 반환하는 것이 주 목적입니다.
- `size()`: 해당 객체가 차지하는 **메모리의 크기**를 반환하는 것이 주 목적입니다.
- C 언어 스타일 문자열(char*)과 달리 `std::string`은 문자열 끝에 `\\0` 문자를 붙이지 않아 `length()`와 `size()`의 결과가 같습니다.

<aside>

🔍 **`length()`와 `size()`의 차이점**
이 두 함수는 둘 다 문자열의 "길이"를 알려주는 것처럼 보이지만, 사실은 조금 다른 의미를 가지고 있습니다.

- **`length()`**: 이 함수는 말 그대로 문자열에 들어있는 **글자의 개수**를 알려줍니다. 예를 들어, "Hello"의 `length()`는 5가 됩니다.
- **`size()`**: 이 함수는 해당 `string` 객체가 **메모리에서 얼마나 많은 공간을 차지하고 있는지**를 알려줍니다. "Hello"의 `size()`도 5가 됩니다.

**언제 무엇을 쓸까?**

- **문자열의 글자 개수가 필요할 때**: `length()`를 사용하는 것이 의도에 더 맞습니다.
- **메모리 사용량이나 객체 크기가 필요할 때**: `size()`를 사용하는 것이 더 정확한 의미를 가집니다.
</aside>

# 빈 문자열 검사하기 — `empty`

`empty()` 함수는 문자열이 비어 있는지(길이가 0인지) 검사하여, 비어 있으면 `true`를, 그렇지 않으면 `false`를 반환합니다. `std::boolalpha`를 사용하면 `true`나 `false`로 출력하고, `std::noboolalpha`로 다시 0 또는 1로 출력되게 할 수 있습니다.

# 문자열 추가하기 — `append`

`append()` 함수는 기존 문자열의 끝에 새로운 문자열을 추가합니다.

- `문자열.append("추가할_문자열")`: 전체 문자열을 추가합니다.
- `문자열.append("추가할_문자열", 문자열_시작_인덱스, 문자_개수)`: 추가할 문자열에서 특정 부분만 지정하여 추가합니다.

<aside>

### 🔍 **`append()` 함수**

**두 가지 주요 사용법:**

1. **전체 문자열 붙이기:**
    
    ```cpp
    std::string str1("Hello");
    str1.append(" World!"); // str1은 이제 "Hello World!"가 됩니다.
    std::cout << str1 << std::endl; // 출력: Hello World!
    ```
    
2. **문자열의 특정 부분만 붙이기:**
    
    ```cpp
    std::string str2("Hello");
    // " World!"에서 인덱스 6부터 1개의 문자만 추가 (즉, '!'만 추가)
    str2.append(" World!", 6, 1);
    std::cout << str2 << std::endl; // 출력: Hello!
    ```
    
</aside>

# 문자열 찾기 — `find`

`find()` 함수는 특정 문자열이나 문자가 나타나는 첫 번째 위치(인덱스)를 반환합니다.

- `문자열.find("찾을_문자열")`: 문자열의 처음부터 탐색합니다.
- `문자열.find('찾을_문자')`: 단일 문자를 탐색합니다.
- `문자열.find("찾을_문자열", 시작_위치)`: 지정된 시작 위치부터 탐색합니다.
- 찾는 문자열을 찾지 못하면 `string::npos` (일반적으로 -1로 정의된 `size_t` 타입 상수)를 반환합니다.

<aside>

### 🔍 **`find()` 함수**

`find()` 함수는 어떤 문자열 안에 특정 글자나 다른 문자열이 "어디에 있는지" 찾아주는 역할을 합니다. 마치 책에서 특정 단어를 찾아 몇 페이지에 나오는지 알려주는 색인(index)과 같습니다.

**주요 사용법:**

1. **전체 문자열에서 찾기:**
    
    ```cpp
    std::string str = "This is an example.";
    size_t pos = str.find("example");
    std::cout << "example found at index: " << pos << std::endl; // 출력: 11
    ```
    
    `str.find("example")`은 "example"이 `str` 문자열에서 처음 시작하는 위치, 즉 인덱스 11을 반환합니다. (공백 포함, 0부터 시작)
    
2. **특정 위치부터 찾기:**
    
    ```cpp
    std::string str = "This is an example.";
    // "is"를 인덱스 4부터 찾기
    size_t pos2 = str.find("is", 4);
    std::cout << "is found at index from 4: " << pos2 << std::endl; // 출력: 5
    ```
    
    "This **is** an example."
    
    - 첫 번째 "is"는 인덱스 2에 있습니다.
    - 두 번째 "is"는 "This is an example"의 "is" 중 두 번째 'i'는 인덱스 5에 있습니다.
    `str.find("is", 4)`는 인덱스 4(`s`)부터 찾기 시작하므로, 두 번째 "is"의 시작 위치인 5를 반환합니다.
3. **단일 문자 찾기:**
    
    ```cpp
    size_t pos3 = str.find('T');
    std::cout << "T found at index: " << pos3 << std::endl; // 출력: 0
    ```
    
    'T'와 같이 한 글자만 찾을 수도 있습니다.
    

**`string::npos`의 의미:**

만약 `find()` 함수가 찾는 문자열이나 문자를 **찾지 못하면**, `string::npos`라는 값을 반환합니다.
`string::npos`는 `size_t` 타입의 상수로, 실제로는 `-1`과 같은 큰 양수 값으로 정의되어 있습니다 (부호 없는 정수 `size_t` 타입이므로 -1은 가장 큰 값으로 표현됨).

```cpp
std::string str = "Hello";
size_t pos_not_found = str.find('k'); // 'k'는 문자열에 없음
if (pos_not_found == std::string::npos) {
    std::cout << "'k'를 찾을 수 없습니다." << std::endl; // 이 코드가 실행됨
}
```

따라서 `find()` 함수의 반환값을 확인할 때는 반드시 `string::npos`와 비교하여 성공 여부를 판단해야 합니다.

</aside>

# 문자열 비교하기 — `compare`

`compare()` 함수는 두 문자열을 비교하여 정수 값을 반환합니다.

- `0`: 두 문자열이 **완전히 같음**.
- `양수`: 대상 문자열이 더 길거나, 일치하지 않는 첫 번째 문자가 사전 순으로 **더 큼**.
- `음수`: 대상 문자열이 더 짧거나, 일치하지 않는 첫 번째 문자가 사전 순으로 **더 작음**.
- 관계 연산자(`==`, `!=`, `<`, `>`)를 사용하여 더 직관적으로 문자열을 비교할 수도 있습니다.

# 문자열 교체하기 — `replace`

`replace()` 함수는 문자열의 특정 부분을 다른 문자열로 교체합니다. 다양한 오버로드된 함수 원형을 제공합니다.

- `string& replace(size_t pos, size_t len, const string &str)`: `pos` 위치부터 `len` 길이만큼의 문자열을 `str`로 교체합니다.
- `find()` 함수와 함께 사용하면 특정 문자열을 찾아 교체하는 작업을 효율적으로 수행할 수 있습니다. `find()`가 문자열을 찾지 못하면 `string::npos`를 반환하므로, 이를 확인하여 `replace()`를 호출해야 합니다.

**가장 기본적인 사용법:**`문자열.replace(시작_위치, 교체할_길이, "새로운_문자열");`

- **`시작_위치 (pos)`**: 문자열에서 교체를 시작할 인덱스(0부터 시작)입니다.
- **`교체할_길이 (len)`**: 시작 위치부터 몇 글자를 교체할지 그 길이를 지정합니다.
- **`"새로운_문자열" (str)`**: 기존 문자열의 해당 위치를 대체할 새로운 문자열입니다.

**예시:**

```cpp
std::string str = "Hello, world!";
str.replace(7, 5, "C++"); // 인덱스 7부터 5글자("world")를 "C++"로 교체
std::cout << "Replaced string: " << str << std::endl; // 출력: Hello, C++!
```

# 와이드 문자열 — `wstring`

`std::string`은 유니코드를 지원하지만, 다국어 처리 시 인코딩을 직접 관리해야 하는 번거로움이 있습니다. C++11부터는 `std::wstring`을 제공하여 와이드 문자(wide character)인 `wchar_t`를 기반으로 유니코드 문자열을 더 직관적으로 다룰 수 있게 합니다.

- `setlocale(LC_ALL, "")`: 프로그램의 지역(locale) 설정을 유니코드 환경에 맞게 변경합니다.
- `L"문자열"`: 와이드 문자열 리터럴을 나타내는 접두어입니다.
- `std::wcout`, `std::wcin`: 와이드 문자열 입출력 스트림입니다.

**유니코드(Unicode)**: 전 세계 모든 문자를 표현하기 위한 표준 문자 인코딩 시스템입니다. 언어나 지역에 따른 호환성 문제를 해결하며, 모든 문자를 고유한 코드 포인트로 매핑합니다 (예: 'A'는 U+0041, '가'는 U+AC00). 다국어 프로그램 개발에 필수적입니다.

<aside>

🔍 **`wstring`이란?**`std::wstring`은 `std::string`과 비슷하지만, 주로 **유니코드 문자열**, 즉 전 세계의 다양한 언어(한글, 일본어, 중국어 등)의 글자들을 더 쉽게 다루기 위해 만들어진 특별한 문자열 타입입니다. `std::string`이 주로 영어와 같은 1바이트 문자를 다루는 데 최적화되어 있다면, `std::wstring`은 2바이트 이상의 와이드 문자를 다루는 데 초점을 맞춥니다.

😊 **`wstring`을 사용하는 이유 (유니코드와의 관계):**

- **유니코드**: 컴퓨터가 전 세계의 모든 글자를 표현하기 위한 통일된 약속(표준)입니다. 예전에는 각 나라마다 컴퓨터가 글자를 저장하는 방식(인코딩)이 달라서, 한국어 문서를 다른 나라 컴퓨터에서 열면 글자가 깨지는 문제가 많았어요. 유니코드는 이런 문제를 해결하고, 모든 글자에 고유한 번호(코드 포인트)를 부여해서 어떤 컴퓨터에서든 똑같이 글자를 볼 수 있게 합니다.
- **`std::string`의 한계**: `std::string`도 유니코드를 저장할 수 있지만, 한글처럼 여러 바이트로 이루어진 문자를 처리할 때 개발자가 인코딩(UTF-8, UTF-16 등)을 직접 신경 써야 하는 복잡함이 있어요. 예를 들어, 한글 한 글자가 `str[0]`처럼 하나의 문자로 처리되지 않고, 여러 바이트로 쪼개져 보일 수 있습니다.
- **`std::wstring`의 장점**: `std::wstring`은 `wchar_t`라는 "와이드 문자" 타입(일반적으로 2바이트 또는 4바이트)을 사용하여 유니코드 문자를 저장합니다. 덕분에 한글 한 글자를 `wstring`에서는 하나의 `wchar_t`로 취급하여 더 직관적으로 다룰 수 있습니다.

**`wstring` 사용 방법:**

1. **지역(Locale) 설정**: `setlocale(LC_ALL, "");` 함수를 호출하여 프로그램의 언어 및 문화권 설정을 시스템 기본값(주로 유니코드 환경)에 맞춥니다. 이는 `wcin`, `wcout`이 올바르게 작동하도록 돕습니다.
2. **`L` 접두어**: `std::string`이 `"Hello"`처럼 일반 문자열을 쓰는 것과 달리, `std::wstring`은 `L"안녕하세요"`처럼 문자열 앞에 `L`을 붙여 "이건 와이드 문자열이야!"라고 알려줍니다.
3. **입출력 스트림**: `std::cout` 대신 `std::wcout`을, `std::cin` 대신 `std::wcin`을 사용합니다.

**예시:**

```cpp
#include <iostream>
#include <string>    // std::string과 함께 std::wstring도 포함됩니다.
#include <locale>    // setlocale 함수를 사용하기 위한 헤더

int main() {
    setlocale(LC_ALL, ""); // 프로그램의 지역 설정 (유니코드 처리를 위함)

    std::wstring korString = L"안녕하세요"; // 와이드 문자열 초기화 (L 접두어)
    std::wcout << korString << std::endl; // 와이드 문자열 출력 (wcout 사용)

    std::wstring name;
    std::wcout << L"이름을 입력하세요: ";
    std::wcin >> name;
    std::wcout << L"환영합니다, " << name << L"님!" << std::endl;

    return 0;
}
```

`wstring`은 다국어 지원이 필수적인 애플리케이션(예: 국제화된 소프트웨어)을 개발할 때 매우 중요한 도구입니다. 이 외에도 UTF-16을 위한 `std::u16string`, UTF-32를 위한 `std::u32string` 등 다양한 유니코드 문자열 형식이 있습니다. 

</aside>
